AHK实现窗口快捷切换

2021-01-16 2 min read # AHK # Toolkit

Win 窗口切换快捷键最优解 - 知乎

原生的切换（alt + tab），最好一点是

  • 快按直接切换窗口
  • 慢按可以在调出之后，不换键就能实现选择。
    所以能够在一个键位实现多种功能
    但缺点是必须一直按住alt

我试图做到：

  1. 如果是两个窗口，那么实现一键切换，即一键实现Alt + Tab
  2. 如果是多个窗口，那么尽量减少窗口选择过程，并且不需要一直按住键盘，即一键实现Ctrl + Alt + Tab
  3. 尽量使用方便并且不干扰现有按键的那些键位，如右侧的Alt或者Ctrl，PrintScreen，左侧的Capslock等
  4. 最好是左手按键，省得右手在鼠标和键盘之间切换

那么最佳选项应该是Capslock：

  • 按一次实现切换窗口
  • 按两次实现窗口可选择
    当然两者会互相干扰，一次的情况要慢一些触发，为两次的情况留下空间。
